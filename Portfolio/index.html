ABOUT ME

안녕하세요! 웹 퍼블리셔 이세련입니다.
좋아하는 분야에 호기심이 넘치고, 탐구심을 가지고 하나씩 파헤치는 과정을 좋아합니다. 웹을 만들며 지켜야하는 모든 규칙에 호기심을 품고 이유를 찾으며 공부한 기록을 가볍게 정리했습니다.

  • 시맨틱 태그, 왜 중요할까?

    HTML

    SEO 최적화를 위해 시맨틱 마크업, 메타 태그, 이미지 alt 속성, 반응형 구조를 적용합니다. a 태그와 button 태그를 목적에 맞게 구분하여 사용하며, img 태그와 background-image의 장단점을 이해하고 상황에 맞게 선택합니다. 시맨틱 태그는 웹표준과 접근성의 첫걸음으로, 모든 기기에서 동일한 정보를 제공하는 웹을 만듭니다.

    퍼블리셔에게 css란?

    CSS

    다양한 동적 효과를 CSS로 구현할 수 있습니다. 가상 요소를 활용한 커스텀 디자인, 변수와 keyframes를 이용한 애니메이션 제작이 가능합니다. flex와 grid로 반응형 레이아웃을 효율적으로 구성하며, float, inline-block, position 속성을 상황에 맞게 사용할 수 있습니다. table 구조로 게시판·FAQ를 구현하고, 가독성 있는 클래스 네이밍으로 가볍고 유연한 웹을 만듭니다.

  • 자바스크립트, 어디까지 알고있니?

    JavaScript

    jQuery와 다양한 라이브러리를 활용하기 위한 로직을 이해하고, UI 기능 구현을 위한 선언문과 실행문을 작성할 수 있습니다. 프레임워크 환경에서도 작업할 수 있는 퍼블리셔를 목표로 JS를 단계적으로 공부하고 있습니다. 또한 효과적인 UX를 구현하기 위해 anime.js, GSAP 등의 애니메이션 라이브러리 활용을 꾸준히 연습 중입니다.

    제이쿼리 마스터했어?

    jQuery

    네비게이션, 탭메뉴, 모달, 팝업, 패밀리사이트, 탑버튼, 스크롤 이벤트 등 다양한 UI 기능을 구현할 수 있습니다. 바닐라 JS와의 장단점을 이해하고 상황에 맞게 선택하여 사용합니다.

  • 너의 공부기록을 보여줘!
